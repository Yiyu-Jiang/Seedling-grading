## 六种骨干网络结构说明

### 1. R_50_3x（ResNet-50, 3x schedule）
- **主干**: ResNet-50，共50层，4个残差阶段（C2-C5），参数量约25M
- **特征金字塔**: FPN（Feature Pyramid Network），多尺度融合输出P3-P7
- **检测头**: FCOS全卷积无锚框检测 + BlendMask实例分割头
- **训练策略**: 3x schedule（约270k iter缩放至本实验40k iter），标准数据增强
- **特点**: 参数量适中，速度与精度平衡，是分割任务的常用基准模型

### 2. R_101_3x（ResNet-101, 3x schedule）
- **主干**: ResNet-101，共101层，比R-50多51层残差块（stage3从6→23个block），参数量约44M
- **特征金字塔**: 与R-50相同FPN结构
- **检测头**: FCOS + BlendMask
- **训练策略**: 3x schedule，与R_50_3x完全相同配置
- **特点**: 更深网络提升语义表达能力，尤其对中大目标分割效果更好，但推理更慢

### 3. 550_R_50_3x（ResNet-50, 550输入分辨率）
- **主干**: ResNet-50（与R_50_3x相同）
- **输入分辨率**: 固定550×550（原始R_50_3x为640-800短边），降低输入尺寸
- **特征金字塔**: 针对550分辨率调整的FPN
- **检测头**: FCOS + BlendMask
- **特点**: 固定低分辨率输入显著加快推理速度（77ms vs 94ms），但小目标精度有所损失（APs=71.4 vs 75.3）

### 4. 550_R50_dcni3（ResNet-50 + DCN, 550输入）
- **主干**: ResNet-50 + Deformable Convolution Networks（DCN）
  - DCN在stage3、stage4的部分卷积层替换为可变形卷积（Deformable Conv）
  - 可变形卷积可自适应学习感受野形状，更好处理形变目标
- **输入分辨率**: 550×550（与550_R_50_3x相同）
- **特征金字塔**: FPN
- **检测头**: FCOS + BlendMask
- **特点**: 在550低分辨率基础上引入DCN，提升对不规则形状目标的建模能力，mAP比550_R_50_3x提升0.43个点

### 5. R_101_dcni3（ResNet-101 + DCN）
- **主干**: ResNet-101 + Deformable Convolution（DCN），最强特征提取器
  - 深层网络（101层）+ 可变形卷积的组合
  - 同时具备深层语义特征和自适应感受野两大优势
- **输入分辨率**: 标准多尺度（640-800短边）
- **特征金字塔**: FPN
- **检测头**: FCOS + BlendMask
- **特点**: 六种配置中**精度最高**（mAP=89.25，AP75=98.95），但推理最慢（101ms），适合高精度离线场景

### 6. RT_R_50（Real-Time ResNet-50, SyncBN）
- **主干**: ResNet-50（轻量化配置）
- **归一化**: SyncBN（Synchronized Batch Normalization）用于多GPU同步
- **检测头**: 轻量化FCOS + BlendMask头，减少特征通道数
- **训练策略**: 4x schedule，专为实时推理优化
- **特点**: 六种配置中**推理最快**（71.8ms），适合实时应用场景，mAP=86.59略低于标准R_50_3x

## 结论

1. **精度最优**: `R_101_dcni3`（ResNet-101 + DCN）在mAP、AP75、APs、APm上均领先，适合对分割精度要求最高的场景
2. **速度最优**: `RT_R_50`（实时模型）推理仅需71.8ms，适合实时或边缘部署场景
3. **平衡最优**: `R_50_3x` 在93.6ms内达到mAP=88.3，是精度与速度的最佳平衡点
4. **DCN增益**: 在同等骨干（R-50或R-101）下引入DCN，mAP平均提升约0.4-0.8个点，代价是推理时间略增（约8ms）
5. **分辨率影响**: 550输入相比标准输入推理速度提升约15-20ms，但小目标精度（APs）下降约4个点
6. **所有模型AP50 > 98.96**：在IoU=0.5阈值下精度接近完美，说明模型对生菜实例的检测定位能力极强
